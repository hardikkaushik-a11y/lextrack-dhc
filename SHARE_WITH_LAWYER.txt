Hi! I've built you a practice management dashboard for your IPR litigation cases.
Here's what you get — no installation, no server, completely free.

───────────────────────────────────────────────────────────────────────────────
WHAT'S INCLUDED
───────────────────────────────────────────────────────────────────────────────

✓ Dashboard — See all your active matters, hearings this week, pending tasks at a glance
✓ Case Lookup — Type any DHC case number → instantly see next date, last date, AI analysis
✓ All Matters — Full table with filters, add/delete cases
✓ Hearings Calendar — Visual calendar with upcoming hearings sidebar
✓ Tasks — All action items across all cases, prioritized (P1/P2), mark done
✓ AI Analysis — Click one button → get strategic insights, action items, documents needed
✓ Documents — Track all pleadings, orders, affidavits in one place

───────────────────────────────────────────────────────────────────────────────
HOW TO USE (3 STEPS)
───────────────────────────────────────────────────────────────────────────────

1. DOWNLOAD THE APP
   • Download: LexTrack-IPR-App.html
   • Open in Chrome/Safari/Firefox
   • That's it. No login. No installation.

2. ADD YOUR CASES
   • Click "+ New Matter"
   • Fill in case number, type, dates, client name
   • Or edit the HTML file to bulk-import 10+ cases at once

3. GET AI INSIGHTS
   • Open any case → Click "✦ AI Analysis"
   • Get instant summary: what happened last date, what's next, what to do
   • (Optional: Add your Claude API key for live AI — ₹200-500/month)
